# Laravel5.5-Japanese-Language-fileset
## Laravel5.5 日本語言語ファイルセット  
1. resources/lang/ja ディレクトリを作成し、その中にこれらのファイルを設置してください。  
2. config/app.php の locale を ja へ変更する
```
'locale' => 'ja',
```
